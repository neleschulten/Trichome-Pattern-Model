# Trichome Pattern Formation Model

This repository contains an interactive Streamlit application to simulate trichome pattern formation in Arabidopsis thaliana.
The model is based on a system of coupled differential equations describing the interaction between activator and inhibitor components (e.g. GL1, GL3, TRY, CPC) and their complexes (AC1, AC2).

## Purpose

This project is part of a Master's thesis in biology education.
The goal is to develop a digital teaching module that helps students understand pattern formation processes in plants through interactive simulation.
